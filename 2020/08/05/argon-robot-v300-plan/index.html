ArgonRobot-V3.0.0 发布计划

ArgonRobot是一个数据爬取和分析的综合管理平台。平台基于 Python3 + Flask + Peewee + Redis 开发。

主要功能如下：

  • 数据抓取（包括定时和手动两种方式）
  • 配置数据执行器
  • 社交平台发布
  • 平台月度/年度运营报告

发布进程如下：

日期 事项
2020.08.10 V3.0.0Beta1 发布 ，冻结新功能
2020.08.16 完善文档
2020.08.21 V3.0.0RC1发布
2020.08.25 V3.0.0 发布
