发表于 2019-04-25 分类于 技术研究 阅读次数： Valine：

本文描述了将 python 源代码文件编译成相应的库文件。

1 文件类型

python开发中文件类型如下：

序号 文件类型 描述
1 .py 源代码文件
2 .pyw 源代码文件
3 .pyi 存根文件，用于类、函数的声明；参见PEP 484
4 .pyc 字节码文档
5 .pyo 和pyc类似，优化后的字节码文件
6 .pyd 库文件，和dll 和 so 文件类似

和dll的区别

关于 dll 和 pyd 的区别可以参见 官方文档 Is a *.pyd file the same as a DLL?。
